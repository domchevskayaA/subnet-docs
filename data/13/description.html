SN13 is Bittensor's data scraping and storage subnet. Developing state-of-the-art intelligence depends disproportionately on the underlying datasets, and SN13 provides the Bittensor ecosystem with diverse, high-quality datasets from a wide array of sources.

How SN13 Delivers Data at Scale

1. Built from the Ground Up

SN13 is a bespoke, Bittensor-native solution to data collection and storage: designed by Bittensor users, for Bittensor users.

2. Integrated for Better Performance

Its integration with the ecosystem allows other Subnets to easily access and deploy vast amounts of disparate data, driving efficiency.

3. Unlocking New Use Cases

The data collected by this subnet is the very fuel that we need to build the next generation of AI. We're committed to making the very best version of this subnet for miners, validators and end users.
